## 项目定位
OPC模式下的人才-企业双边匹配平台：通过"红点觉醒→认知建设→真实任务→AI画像→精准匹配"闭环，替代传统简历招聘。
- 人才端永久免费，匹配成功后向企业收费
- MVP阶段零开发，全部用现有工具（飞书+Coze）

## 内核决策（2026-04-15 锁定）

### 1. 引擎是人，不是企业
- 人找不到方向是核心问题，企业匹配是放大器
- 未来AI时代的趋势：OPC与OPC在组织层面的协同，不是传统雇佣
- 以人做出发点，给人最适合他长处和优势发挥的企业任务场景

### 2. 红点定位卡是必要条件，但是渐进式的
- 不是"先有红点才能接任务"，而是边做边找
- 三个阶段：初级红点认知 → 过程中不断优化打磨 → 达到明确状态
- 底层飞轮：**通过匹配来定位**，不是先定位再匹配
- 评估模型中"红点清晰度"是动态指标，随任务完成不断上升

### 3. 能力数据是假设待验证，但方向正确
- 任务成果反向验证能力
- 任务过程帮助人辨别自己的优势
- 终极形态：比简历更可信的动态能力档案

### 关键洞察
- 正反馈循环强度 = 红点清晰度的加速度（不是笼统的"越做越好"）
- 红点越清晰 → 匹配越精准 → 企业满意度越高
- 核心竞争力：红点方法论 + 企业信任网络的双壁垒

### 人才数据集模型（Talent Dataset）
向企业展示的不是简历或单一评分，而是四维数据集：
1. **人生红点** — 他要去哪（方向+动力+清晰度）
2. **能力画像** — 他擅长什么/不擅长什么（任务验证过的）
3. **加速度** — 他的成长有多快（跨任务进步曲线）
4. **成长轨迹** — 他怎么走到这里的（实际交付物+评分+复盘）

### 商业模型（2026-04-15 最终锁定）

**Redpoint 不是招聘平台，是带真实任务实践的成长服务。**

| 维度 | 定义 |
|------|------|
| 第一收入 | 人才为自己的成长付费（1500-2000元） |
| 核心产品 | 课程+真实企业任务实践闭环（不是纯听课） |
| 留存方式 | 课程内容+服务配套（前7天即时价值） |
| 第二收入 | 企业端：为任务结果付费 + 为人才推荐付费 |
| MVP验证 | 人是否愿意为"在真实实践中成长"付费 |
| 与知识付费的区别 | 真实企业任务作为实践闭环，不是"听完课就完了" |

### 企业端用户画像
- **HR：** 人才推荐/精准匹配场景（对应用户推荐付费）
- **CEO/业务负责人：** 直接推出企业所需任务（对应任务实现付费）

### OPC定位澄清（2026-04-15 锁定）
**OPC 是业务本身的运营形态，不是用户的身份。**

- 东东以 OPC 形式开展这个业务
- AI 完成 90% 的工作（课程交付、评估、匹配、运营）
- 东东负责 10% 的核心决策和资源连接
- 平台内的人是"成长者"，不是 OPC
- 这意味着不需要为用户建立 OPC 身份认同、法律框架、税务体系

## 核心架构
- **3层+1飞轮培养体系：** 唤醒（红点）→发现（优势）→认知（商业+德鲁克）→实践（真实任务）+ 正反馈螺旋

### 目标用户（2026-04-15 明确）
**两类人群：**
1. **创业者和管理者** — AI时代持续学习、精进、成长的路径（核心付费人群，1500-2000元）
2. **大学生** — "大学生的翱翔计划"，私域性质，价格优惠很多（同套课程，类似苹果大学生优惠）

### 东东的AI能力版图（2026-04-15 明确）
1. **一堂AI云南中心** — AI+创业方法论培训、AI工具实操、创业者社群
2. **硅创联昆明中心** — AI产业资源对接、项目孵化、人才生态
3. **四川大学商学院合作（推进中）** — 引入学术资源、联合课程、校企人才通道
4. **长期目标** — 建立云南本土AI及OPC社群，持续赋能合作平台
5. **运营形态** — OPC，AI 90% + 东东 10%
