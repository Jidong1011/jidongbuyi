# 2026-04-15 记忆日志

## 新项目：人才精准匹配平台

### 项目定义
- **本质**：用"免费课程+真实任务"做漏斗，把大学生和成人从学习者变成可评估的人才，匹配给企业
- **三引擎架构**：
  1. 🧠 认知引擎（人生红点课）— 帮人找到"我是谁、我要去哪"
  2. 🛠️ 实践引擎（反馈分析法）— 企业发单，人才做真实任务，建立能力数据
  3. 📊 数据引擎（AI匹配）— 基于真实行为数据匹配，不是简历
- **飞轮**：免费课程 → 人才涌入 → 企业发单 → 完成任务 → 能力画像 → 精准匹配 → 成功案例 → 更多人才和企业
- **目标用户**：大学生（大一/二试错）+ 成人就业者（高密度重新定位）
- **商业逻辑**：人才端免费 + 企业发单免费 → 匹配成功后向企业收费

### 人才评估算法：红点×能力 双轴模型
- **核心动力（权重40%）**：人生红点清晰度15% + 使命笃定度15% + 内在驱动力10%
- **基础素质（权重60%）**：分析性思维10% + 动手实践能力10% + 好奇心指数8% + 主动学习力8% + 创造力8% + 协作沟通力8% + AI工具协作力8% + 韧性指数10%
- **理论依据**：WEF 2025 + SREB 2025 + arXiv AI Pyramid 三篇权威来源
- **四象限**：明星（高动力高能力）/ 潜力（高动力低能力）/ 实干（低动力高能力）/ 待确认（低动力低能力）
- **匹配逻辑**：硬匹配×0.3 + 软匹配×0.4 + 动力匹配×0.3

### 竞品研究结论
全球没有完整做过"认知→实践→数据→匹配"闭环的平台：
- **技能评估层**：HackerRank/CodeSignal/Pymetrics — 只评估不培养
- **学习加速层**：Pathrise/Handshake/Degreed — 培养与就业割裂
- **零工市场层**：Upwork/Fiverr — 做完就散，无能力画像
- **AI匹配层**：Eightfold AI/Gloat — 面向企业内部
- **护城河**：闭环（认知→实践→数据→匹配），一个平台全走完
