## 关键决策

### Redpoint AI人才画像MVP确定
- **场景**：AI自动读简历，生成人才评估报告
- **痛点**：人工2小时 → AI 10分钟
- **验证标准**：10份画像，≥6份可用，<10分钟/份
- **本周行动**：收5份简历 → 搭Coze工作流 → 测3个case

### 一页纸核心理念
一页纸是核心交付物，必须包含：
1. 一句话场景定义
2. 四大维度验证（有容错/有方法/能验证/有价值）
3. 量化业务目标
4. 清晰边界（不做什么）
5. 本周可执行动作

## 方法论要点

### STAR模型
- **S-Self**：个人痛点（重复性工作、信息处理）
- **T-Team**：团队瓶颈（标准不一、专家时间）
- **A-AI**：AI能力匹配（文本生成、信息提取、分类标签）
- **R-Reality**：现实约束（数据、容错、成本、时间）

### 线性推演
把复杂业务按时间线拆解为节点：
```
节点 = 输入 + 处理逻辑 + 输出
```
识别高AI可行性节点，确定P0/P1/P2优先级
