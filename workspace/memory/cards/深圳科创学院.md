# 深圳科创学院 速查卡

> **一句话本质**：李泽湘把30年孵化大疆、云鲸的经验打包成一个"硬科技创业军校"，从训练营筛选到入孵孵化到资金支持，一站式把理工科学生变成公司创始人。

## 核心三要素
1. **李泽湘生态**：200+孵化企业（含4家上市、10+独角兽），完整供应链+导师+投资网络
2. **从Day 1就创业**：不教理论，直接从市场需求出发组建团队、定义产品、验证原型
3. **一站式全链路**：50万-500万资金 + 实验室 + 供应链 + 出海 + 投融资，覆盖创业全流程

## 一句话类比
传统大学是体育课（学规则、跑跑步），InnoX是体校（直接给你教练、场地、比赛机会，目标是签职业合同）。（失效边界：体校训练的是已有技能，InnoX面对的是创造从未存在过的产品，不确定性高得多）

## 关键出处
- 福布斯2025年报道称其为"中国版YC"，但覆盖从0到1的全过程
- 于盈（副院长）：硅谷硬件成本是深圳的5-30倍，但深圳迭代速度是美国的5-10倍
- 李泽湘体系数据：孵化企业存活率80%，独角兽率15%

## 何时引用
- 了解中国硬科技创业教育的最新模式
- 对比深圳vs硅谷的硬件创业成本与速度
- 寻找硬科技方向的早期创业支持资源
- 设计创业培训/孵化器项目时参考

- Y Combinator（对标，但阶段和模式不同）
- 大疆/云鲸/正浩（李泽湘体系代表企业）
