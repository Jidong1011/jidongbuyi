# AI个人复利系统 - Garry Tan 速查卡

> **一句话本质**：YC总裁Garry Tan的AI系统两个月涨10倍——让AI有记忆、有流程、有检查，每次用完都在滚雪球，而大多数人的AI每次从零开始

## 核心三要素

1. **无状态困境**：传统AI每次对话都从零开始，不记得你是谁、做了什么
2. **三层复利**：记忆层（存储）+ 流程层（Skill）+ 检查层（避坑），1+1+1 >> 3
3. **Meta-Meta-Prompting**：不只是写提示词，而是建立让AI自己优化的系统

## 一句话类比

一次性AI像**租房子**——每次搬家重新买家具，所有投入都是沉没成本。
复利AI像**买房子**——每次装修都在增值，五年后一切都还在，而且更多。

失效边界：问"今天天气"不需要复利；复利在长期关系、复杂项目中才体现。

## GBrain核心机制

```
读取(对话开始时加载记忆)
    ↓
对话(带着记忆和你交流)
    ↓
写入(对话结束后提取关键信息存入)
    ↓
知识库进化(下次对话更懂你)
```

## 如何开始

1. 本周：创建memory/目录，每天写日记，对话开始让AI先读
2. 本月：识别3个重复工作流，写成检查清单/流程
3. 持续：每次AI犯错记录避坑指南，定期回顾更新
