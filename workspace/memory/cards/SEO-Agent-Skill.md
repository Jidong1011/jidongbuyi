# SEO Agent Skill实践 速查卡

> **一句话本质**：把SEO的重复流程（关键词挖掘、网站审计、内容创作）打包成AI可执行的Skill包，每次执行成本仅几毛钱，让AI变成你的SEO实习生

## 核心三要素

1. **两步走**：拆解SOP → 识别可Skill化的流程（检查逻辑固定、标准化的部分）
2. **细分比通用好**：Listicle/Comparison/How-to分别做Skill，一个Skill覆盖所有范本结果每个都做不好
3. **好的Skill = 经验蒸馏 + 自我学习**：把SOP写成代码 + 建立反馈机制让Skill进化

## 一句话类比

Agent Skill就像餐厅的"标准菜谱"——把你做SEO的经验写进去，新来的实习生（AI）照着做就能做出80分的菜。

失效边界：新场景（新SEO需求）需要新菜谱；Skill也需要反馈和迭代才能保持质量。
