# Ruflo多Agent编排平台 速查卡

> **一句话本质**：像带团队一样指挥一群AI Agent——60+专业Agent分工协作、自学习进化、跨设备联邦协作，Token成本降低75%

## 核心三要素

1. **多Agent协作**：60+专业Agent协调工作，Queen-Worker层级调度，任务分解后并行执行
2. **SONA自学习**：Agent从历史任务中持续学习，越用越聪明
3. **联邦协作**：多台设备上的Agent可以跨机器安全通信，协同作战

## 一句话类比

单个Agent像一个万能厨师（每样60分），Ruflo像餐厅厨房（分工协作，每道菜90分）。

失效边界：煎个鸡蛋（简单任务）叫一群人来反而更慢；简单任务一个Agent就够。
