# 《2028全球智能危机》速查卡

> **一句话本质**：如果AI完全按乐观预期发展，可能引发2028年全球智能危机，核心是AI产出不进入消费循环的"幽灵GDP"和恶性循环的"人类智能替代螺旋"

## 核心三要素
1. **幽灵GDP（Ghost GDP）**：AI创造了大量产出，但这些产出没有变成工资发给工人，而是留在了老板手里，所以虽然GDP数字好看，但大家没钱消费
2. **人类智能替代螺旋**：AI越聪明→公司裁员越多→失业的人消费越少→公司利润下降→公司更努力地用AI→AI更聪明...这是一个没有自然刹车的恶性循环
3. **习惯性中介护城河消失**：很多公司赚钱靠的是"人类懒惰"（比如你习惯用某个APP点外卖），但AI没有习惯，它会每次找最便宜的，所以这些公司的护城河就消失了

## 一句话类比
想象学校里有个超级学霸（AI）能完成所有作业，老师辞退了其他学生（白领），但其他学生都没钱买午餐了，学校食堂（消费经济）生意越来越差，最后连学霸的工资也发不出来了。

**类比失效边界**：这个类比简化了经济循环的复杂性，忽略了政府干预、新产业创造等因素
