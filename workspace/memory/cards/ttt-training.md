# TTT培训模式 速查卡

> **一句话本质**：TTT是把"会做"变成"会教"的系统方法论——课程设计（编）+ 授课交付（演）+ 效果评估（评），底层是成人学习理论。

## 核心三要素
1. **课程设计（编）**：用ADDIE模型把知识变成结构化学习路径
2. **授课交付（演）**：30%讲授+70%互动/实操，让学员真正学会
3. **效果评估（评）**：柯氏四级（反应→学习→行为→结果），重点关注L3行为改变

## 一句话类比
厨师长（自己做菜好吃）≠ 烹饪培训师（让别人也能做好菜）。TTT就是弥补这个差距的方法论。（失效边界：创造力培养场景不适用）

## 关键出处
- ADDIE模型：1975年美国佛罗里达州立大学为美军开发
- 柯氏四级评估：Donald Kirkpatrick 1959年提出
- 成人学习理论：Malcolm Knowles 六原则

## 何时引用
- 设计新课程时 → 用ADDIE五步流程
- 授课前 → 复习控场技术和开场方法
- 评估培训效果 → 用柯氏四级模型
- 规划个人成长 → 对照四阶段成长路径

## 关联主题
- 课程开发 → ADDIE/PRM/4T模型
- 成人学习 → Knowles理论/建构主义
- 培训评估 → 柯氏四级/ROI分析
- AI赋能培训 → 智能课程设计工具
