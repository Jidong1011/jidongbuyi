# 马斯克Samson峰会演讲 速查卡

> **一句话本质**：2026年5月18日，马斯克凌晨2:30视频连线以色列，用10分钟预言：10年后开车成小众爱好、SpaceX估值2万亿即将IPO、Neuralink今年让盲人复明

## 核心五要素

1. **自动驾驶终局**：10年后90%里程由AI驾驶，人类开车成"情怀行为"
2. **人形机器人**：汽车是"四个轮子的机器人"，未来机器人数量超过人类
3. **以色列创新**：人均创新能力世界第一，FSD 2026年落地以色列
4. **SpaceX IPO**：估值超2万亿美元，可能本周提交招股书，史上最大IPO
5. **Neuralink**：今年完成首例盲人复明手术，长期目标是认知增强

## 一句话类比

马斯克这次就像一个连发五枚"科技核弹"的预言家——每个都够上头条，但他历史上有"画饼延迟"的前科（2014-2019年FSD预言全没按时兑现），这次有实际运营数据支撑，可信度比以前高一些。

- Samson峰会：以色列特拉维夫，第9届国际智能出行峰会
