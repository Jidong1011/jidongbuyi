# Anthropic组织文化速查卡

> **一句话本质**：Anthropic通过极端聚焦、使命驱动和低ego文化，构建了AI行业最稳固的组织，在人才争夺战中实现了80%的留存率

## 核心三要素
1. **极端聚焦战略**：只做coding，放弃多模态等所有其他方向，"孤篇压全唐"
2. **使命驱动文化**：所有员工真心相信"AI可能拯救或毁灭世界"，愿意为此放弃股票、高薪
3. **低ego、高信任**：没有明星文化，没有职位等级，所有人都可以挑战CEO，财务数据透明

## 一句话类比
Anthropic就像一个目标明确的学霸俱乐部，所有人只专注于数学竞赛，互相帮助，没人想当"最聪明的那个"，只想让整个小组赢；而OpenAI就像一个什么都想做的超级明星团队，结果内耗严重。

**类比失效边界**：这个类比简化了公司治理的复杂性，忽略了商业竞争、技术路线等因素
