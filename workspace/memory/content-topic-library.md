## 风格指南（Writer 子agent 必读）

1. **字数**：800-1500字（日更无需长文）
2. **标题**：具体、有悬念、能让人想点开
3. **开头**：用一个具体的场景/数据/反常识切入，不要铺垫
4. **结构**：观点+论据+行动号召（3段式）
5. **禁止**：
   - "总的来说"、"值得注意的是"、"让我们来探讨一下"
   - "在这个快速变化的时代"
   - 没有数据/案例支撑的"趋势分析"
   - 换个名字也能成立的建议
6. **必须有**："可带走的东西"——认知框架、可复制方法、或情感共鸣
7. **语气**：像跟一个聪明的朋友聊天，不像写论文
