# 一堂（YiTang）平台背景研究

> 研究日期：2026-04-13
> 来源：网络公开信息

## 平台概况

**一堂**是由许楚（Truman）创办的全国性创业学习平台。

### 创始人背景
- 许楚，以太创服合伙人
- 2016年加入以太创服，参与搭建早期创业者融资平台，累计帮助上千个项目完成0到1起盘
- 见证了数千位创业者的基础性、常识性失败

### 发展历程
- **2018年**：启动"一堂"教育创新业务，搭建商业规律研究团队
- **2022年**：提出"推动科学创业"愿景，招募150位专业人士、3500位共建者
- **2024年**：发布"创业地图"和"管理地图"，提出本土原创商业和管理理论体系
- **2025年**：累计43万人听过课，全球十几个国家和地区，累计复盘作业字数过1.7亿

### 核心方法论：一堂五步法
| 里程碑 | 内容 |
|--------|------|
| 第一里程碑 | 需求（需求四字诀） |
| 第二里程碑 | 解决方案（产品内核） |
| 第三里程碑 | 商业模式（单元模型） |
| 第四里程碑 | 增长（增长周期） |
| 第五里程碑 | 壁垒（护城河模型） |

### 出版物
- 《里程碑》（许楚著，人民邮电出版社，2025年11月）— "中国人自己的商业教科书"
- 30+万人听课，课程评分超9.8分

### 云南创业教育生态（竞品/伙伴参考）
- **西山区AI青创空间**：云南省首个OPC社区，聚焦AIGC及AI技术场景落地
- **保山咖啡产业创业孵化示范园**：围绕咖啡产业的省级创业孵化示范园
- **云南省"创业云南"建设**：创业青年达256万人，248个创业载体
- **昆明已有三个OPC玩家**：AIGC联盟（培训）、西山青创（孵化）、同橙（技术平台）

## 一堂云南学习中心的定位建议

### 东东的定位
- 一堂云南学习中心 = 共创实验室（面向创业者）
- 已有场地，正以共创形式开展活动
- 与纪东不已IP独立运营，但合作协同

### 哆啦dora的观察
1. 一堂是全国性品牌，在创业教育领域有强认知。与一堂合作能借势品牌，但需要注意：一堂的方法论是"通用商业方法论"，而东东的差异化在"特色产业场景"
2. "共创实验室"的定位很好——不是被动授课，而是主动共创，这与纪东不已"实践派"的调性一致
3. 关键问题：一堂云南学习中心是一堂官方授权的校友服务中心？还是东东基于一堂方法论自建的本地化平台？这决定了很多运营细节
