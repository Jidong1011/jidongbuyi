## 一、使命体系

### 总使命
**光亮自己，照亮他人——在产业生态中为创业者和超级个体创造成功舞台。**
- 判断标准：每件事先问——是在"光亮自己"还是"照亮他人"？都不沾就不做
- 思想底座：德鲁克管理哲学（成果在外部 / 知识工作者自我管理 / 贡献者思维）

### 三层使命金字塔
1. **个人使命** — 成为AI时代超级个体范本（1人+AI = 5-10人团队职能）
2. **组织使命** — 六个可控主体各归其位、各尽其能
3. **产业使命** — 推动白芸豆全产业链的平台化、数字化、品牌化、国际化

### 六个可控主体
| 主体 | 定位 | 优先级 |
|------|------|--------|
| 天保桦生物 | 制造和提取（产业根基） | 持续运营 |
| 玲素新零售 | C端品牌和会员制零售 | 持续运营 |
| 白芸豆产业运营公司（待注册） | 平台/仓储/数据（数字化中枢） | 最高优先级 |
| 云南交个朋友电子商务 | 服务和咨询 | 当前最优先激活 |
| 纪东不已（海南）投资 | GP和投资 | 长期 |
| 云南唯恒供应链 | 贸易和供应链备用 | 备用 |

### 三大飞轮
- 飞轮一：白芸豆全产业链 → 产业使命
- 飞轮二：AI社群 → 个人+组织使命的技术支撑
- 飞轮三：创投基金 → 产业使命的资本放大器
- **关系**：咬合的，不是平行的

## 二、定位体系

### 核心定位
**纪东不已 = 产业架构师 + AI实践者 + 内容创作者**
- 产业架构师是根
- AI实践者是方法（龙宫操作系统是核心产出）
- 内容创作者是放大器

### 三维度差异化
1. 白芸豆产业 → 平台总设计师（不与上下游抢利益）
2. AI个体创业 → 特色产业OPC加速器（产业场景对接，非通用培训）
3. 知识付费 → 实践派日更作者（记录过程，非贩卖方法论）

### 不做什么
- 不做通用AI培训
- 不做综合孵化
- 不做技术平台
- 不亲自做C端品牌运营
- 不做重资产投入（Phase 1只有3万预算）

## 三、IP策划

### IP矩阵：双核心驱动
**主IP：纪东不已**
- 载体：微信公众号"纪东不已"
- 四大内容支柱：产业认知 / AI实践 / 读书心得 / 灵感反思
- 节奏：每日一更

**付费IP：每日一角**
- 定价：年费36.5元（一天一角）
- 定位：持续陪伴型知识产品（不是课程，是思考过程）
- 内容差异：公众号发成品，每日一角发过程（厨房 vs 餐厅）
- 协同：公众号引流 → 每日一角转化 → 反哺公众号

### 品牌视觉
- 主IP：深色底(#09090d) + 金色强调(#d4a843)
- 每日一角：暖白色调 + 金色描边 + 日记体

## 四、运营目标与计划

### Phase 0（4月）：基础建设
- IP视觉体系确认 + 公众号视觉升级
- 日更SOP建立
- 每日一角MVP设计
- 云南交个朋友电子商务激活
- KPI：日更30天不断更

### Phase 1（5-7月）：验证期
- 日更持续 + 每日一角正式上线（首批100付费用户）
- 硅创联昆明中心Phase 1（20会员+5场景对接）
- 白芸豆产业运营公司注册
- KPI：日更90天 + 每日一角100人 + OPC会员20人

### Phase 2（8-10月）：成长期
- 龙宫系统自动化>50%
- 每日一角500人
- 公众号月增长>5%
- 产出第一个方法论产品
- KPI：每日一角500人 + 龙宫自动化>50%

### Phase 3（11月-次年1月）：扩展期
- 每日一角收入覆盖内容生产成本
- 硅创联收入覆盖运营成本50%
- 年度复盘
- KPI：年化收入覆盖成本
