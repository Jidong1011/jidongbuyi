## 资产类型说明

- **速查卡**：一句话本质 + 核心三要素 + 一句话类比（所有Deep-Dive必创建）
- **检查清单**：多步骤操作的自检清单
- **话术模板**：对外分享/写作时的标准句式
- **决策框架**：面对选择时的决策标准
