# 一堂AI第一课：核心要点提炼

> **说明**：本文提炼自2026年5月24日一堂内部分享会（"AI第一课"），聚焦"双三角模型"的核心框架、能力段位与未来展望。

## 核心观点：AI时代的教育与竞争力

一堂认为，在AI时代，教育的价值不是削弱，而是以**判断力**为核心的形式变得更稀缺。执行性工作将被AI大量替代，但**问题定义、审美、框架构建、决策判断**等能力，将成为人与人之间、企业与企业之间真正的差距所在。

## 核心理论：双三角模型

这是回答"**如何形成有竞争力的AI应用**"的框架。

### 模型结构
*   **左三角（人类侧）**：问题 → 审美 → 创造力
*   **右三角（AI侧）**：场景 → 数据 → 迭代
*   **核心关系**：人和AI构成一个动态的、辩证统一的螺旋上升系统。人的判断力驱动AI，AI的反馈反过来提升人的认知。

### 模型价值
1.  **系统性思维**：避免陷入工具迷信或单一环节优化。
2.  **归因清晰**：AI产出不好，本质是人的“三角”（问题定义、审美、创造力）或数据问题。
3.  **竞争力标尺**：能用双三角拆解和构建的工作流，才是能形成壁垒、超越平均水平的AI应用。

## 能力段位：五个层级

| 段位 | 核心特征 | 关键能力 |
| :--- | :--- | :--- |
| **L1 初步理解** | 理解框架价值 | 能用双三角解释复杂问题，建立系统性认知。 |
| **L2 相信体系** | 动手拆解学习 | 能相信并反向拆解优秀AI应用背后的双三角支撑。 |
| **L3 全面动手** | 掌握Feature思维 | 理解AI工具分层，能通过“口喷”等方式，基于Feature进行设计和协作。 |
| **L4 带领重构** | 成为顾问/讲师 | 能运用双三角，带领他人或团队进行AI项目设计与辅导。 |
| **L5 组织建设** | 重构研究体系 | 能基于双三角，构建方法论体系、案例库，甚至新的组织分工模式。 |

## 关键方法论：口喷基本功

*   **定义**：在与AI协作时，**不依赖预先编写的详细提示词，而是通过即时的口语（语音）输入，快速、密集地向AI传达需求、纠正目标、控制节奏、提供反馈**。
*   **心法**：**心里装着双三角**。在口喷时，实时判断AI产出在哪个维度（数据？审美？流程？）存在不足，并即时“喷”入相应的修正指令。
*   **价值**：是最小、最可练习的双三角工作循环。能极大提升人机协同的效率和深度，是AI时代的核心基本功之一。

## 未来展望

1.  **一堂的演进**：从“五步法”体系，走向“五步法 + 双三角”的双轮驱动。将系统化地建设双三角的方法论、案例库与工具生态。
2.  **业务形态**：从教育（教人），可能扩展到基于AI Partter的**咨询**（人+AI教练服务）和**智能组织构建**（多Agent协作）。
3.  **行业愿景**：积累各行业高质量的AI落地案例，用双三角打通认知，形成可复用的“最佳实践”资产库，最终探索“AI for Business”的全球最佳实践。
