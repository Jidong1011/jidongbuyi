# OPC定义

## 核心事实
OPC = One Person Company = 一人公司。不是"一个人干所有事"，而是"一个人决策+AI执行+外部协作"的新型创业组织形态。

## 详细信息
- **核心特征：** 一个人+AI = 完整的业务能力
- **运营模式：** 轻资产、低风险、快速验证
- **与传统创业区别：** 不追求规模，追求效率和利润率
- **AI的角色：** 承担90%执行性工作，释放人做判断和整合

## 背景与上下文
东东的Redpoint项目以OPC形式运营，是AI原生组织的实践原型。
