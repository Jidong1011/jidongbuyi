# Redpoint人才匹配逻辑

## 适用场景
- 企业任务匹配人才
- 人才画像构建
- 匹配算法设计

## 问题描述
如何将人才特质与企业任务需求进行精准匹配？

## 解决步骤

### 步骤1：企业任务分析
提取关键要求：
- 必备技能
- 软性要求（沟通、协作等）
- 动机偏好（成长型/稳定型等）

### 步骤2：四维人才画像
| 维度 | 数据项 | 权重 |
|------|--------|------|
| 能力维度 | 技能、经验、证书 | 30% |
| 动机维度 | 目标、价值观、驱动力 | 25% |
| 行为维度 | 过往行为、决策模式 | 25% |
| 潜力维度 | 学习能力、适应性 | 20% |

### 步骤3：匹配度计算
```
匹配度 = 能力匹配×0.3 + 动机匹配×0.25 + 行为匹配×0.25 + 潜力匹配×0.2
```

### 步骤4：输出结果
- 匹配度评分（0-100）
- 优势匹配点
- 潜在风险点
- 推荐理由

## 变体与例外
- **初级匹配：** 只看能力维度
- **深度匹配：** 四维全量评估
