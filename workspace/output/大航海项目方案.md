## 一、项目总览

### 1.1 这件事的本质

中国体重管理市场规模超千亿，但白芸豆品类有一个致命问题——**全是智商税产品，没有一个品牌真正用临床数据说话**。玲素®是目前唯一拿到蓝帽子（国食健注G20230917）的白芸豆保健品，有真实临床数据支撑（35天减重3.51kg），但品牌认知为零。

我们要做的事情很简单：**用3个月时间，从零开始跑通一个最小商业闭环，产生真实现金流。** 不是做品牌策划，不是写商业计划书，而是真刀真枪地卖货、获客、积累数据。

### 1.2 核心目标

| 指标 | 数值 | 备注 |
|------|------|------|
| 总GMV | ≥10万元 | 真实交易，不含内部测试 |
| 种子用户 | ≥100人 | 有复购行为的活跃用户 |
| 复购率 | ≥30% | 首批种子用户30天内复购 |
| 内容产出 | ≥50篇真实体感记录 | 分布在小红书/抖音/微信生态 |
| 私域沉淀 | ≥500人 | 微信社群+小程序注册用户 |

### 1.3 成功标准

**及格线（终点奖）：** GMV达到10万，跑通"内容→获客→成交→复购"闭环，有可复制的方法论。

**优秀线（MVP大奖）：** GMV超过10万，形成至少1个可规模化的获客渠道，复购率≥30%，团队能力模型清晰可复制到其他产品。

### 1.4 为什么是现在

- 蓝帽子保健品审批趋严，新进入者门槛极高，玲素的合规壁垒是护城河
- "抗糖""控卡"概念热度持续，消费者教育成本正在下降
- 一堂大航海提供外部压力和资源，强制3个月出结果
- 东东有健康行业资源和一堂社区人脉，这是冷启动的杠杆

## 二、产品与市场分析

### 2.1 产品核心卖点（必须一字不差地记住）

**一级卖点（30秒电梯演讲）：**
"中国唯一蓝帽子白芸豆保健品。不是普通代餐，不是概念添加——临床验证35天减重3.51kg，一粒阻断66%淀粉吸收。"

**二级卖点（3分钟详细版）：**
1. **蓝帽子认证（国食健注G20230917）**——国家市场监督管理总局批准，意味着安全性和功效性都经过严格审查。市面上99%的白芸豆产品是普通食品，没有功效保证。
2. **75%高纯度α-淀粉酶抑制剂**——多数竞品纯度不到30%，有效成分含量天差地别。
3. **10000 U/g活性单位**——用活性说话，不是用含量说话。高纯度+高活性=实际效果的天壤之别。
4. **专利提取工艺**——不是买原料贴牌，是自有专利从云南云龙2300-2700米高海拔基地直采，全产业链可控。
5. **临床数据支撑**——35天减重3.51kg，不是实验室推测，是真人临床实验结果。

### 2.2 目标用户画像

**核心用户（优先打透）：**

| 画像 | 特征 | 痛点 | 触达渠道 |
|------|------|------|----------|
| 微胖白领女（25-35岁） | 久坐、外卖多、BMI 24-28 | 想瘦但没时间运动，试过很多方法都反弹 | 小红书、抖音 |
| 产后妈妈（28-40岁） | 产后体重难恢复、时间碎片化 | 担心药物影响哺乳，需要安全方案 | 亲子社群、微信 |
| 应酬型中年男（35-50岁） | 饮酒多、碳水摄入高、尿酸偏高 | 大肚子、体检指标亮灯 | 朋友圈、同事推荐 |
| 健身减脂人群（20-35岁） | 有运动习惯、关注营养学 | 想提高减脂效率，需要"作弊餐"救急 | 健身房、健身社群 |

**排除用户：**
- 追求极端减重的人（我们不是减肥药）
- 对保健品完全排斥的人（教育成本太高）
- 18岁以下未成年人（法规限制）

## 三、商业模式

### 3.1 商业闭环

```
真实体感内容 → 社交媒体传播 → 用户关注/私信 → 引导至私域
    → 小程序商城下单 → 体感跟踪服务 → 真实复购 → 新体感内容
```

**闭环的关键环节：**
1. **内容必须是真实的**——不是广告，是真实的体感记录（体重变化、饮食记录、感受）。真实性是唯一护城河。
2. **私域是成交主阵地**——不要把鸡蛋放在抖音/小红书的算法篮子里。公域获客，私域成交。
3. **小程序商城是基础设施**——不用跳转，不用截图，微信生态内完成闭环。
4. **体感跟踪是复购引擎**——不是卖完就不管了。通过体感打卡、体重记录、社群互动，让用户感受到被关注，自然产生复购。

### 3.2 单元经济模型（关键！）

以标准装（128元/盒，毛利率约50%）为基准：

| 指标 | 数值 | 说明 |
|------|------|------|
| 客单价 | 128元 | 首单标准装 |
| 毛利 | 64元 | 50%毛利率 |
| 获客成本（CAC） | ≤30元 | 内容获客为主，极低成本 |
| 首单净利 | 34元 | 128-64(COGS)-30(CAC) |
| 3个月复购率 | 30% | 目标值 |
| 复购客单价 | 200元 | 升级为周期装 |
| 3个月LTV | 128+200×0.3 = 188元 | |
| LTV/CAC | ≥6倍 | 极健康 |

**要达到10万GMV：**
- 如果纯靠新客：需要781个新客（128元×781≈10万）
- 如果新客+复购：需要500个新客 + 150个复购（500×128+150×200≈9.4万，加上体验装基本达标）
- **最优路径：** 第一个月冲500新客，第二三个月靠复购+拉新双轮驱动

### 3.3 增长飞轮

```
第一阶段（W1-W4）：冷启动
  10-20个种子用户（东东朋友圈+一堂社区）→ 真实体感记录 → 产出第一批内容

第二阶段（W5-W8）：内容扩散
  50篇体感内容铺小红书/抖音 → 算法推荐带来自然流量 → 转化率优化
  种子用户产生口碑 → 社群裂变 → 累计100+付费用户

第三阶段（W9-W13）：飞轮加速
  内容矩阵形成规模效应 → 外部KOC主动合作 → 渠道官拓展OTC
  复购数据验证 → 周期装/订阅制上线 → 客单价提升
```

## 八、风险与对冲

### 8.1 风险矩阵

| 风险 | 概率 | 影响 | 对冲策略 |
|------|------|------|----------|
| 种子用户招募困难 | 中 | 高 | 降低门槛（免费体验装）、东东亲自邀请、一堂社区资源 |
| 内容不爆（零互动） | 高 | 中 | A/B测试不同内容形式、学习对标账号、找KOC合作 |
| 复购率低 | 中 | 高 | 加强体感跟踪服务、社群运营、个性化推荐 |
| 供应链断货 | 低 | 高 | 提前备货、与厂家确认库存周期 |
| 团队成员退出 | 中 | 中 | 每个角色有备选人选、东东可兼任、关键知识文档化 |
| 竞品低价冲击 | 低 | 中 | 强调蓝帽子差异、不打价格战、用信任赢 |
| 平台限流/封号 | 中 | 中 | 多平台布局、避免违规话术、真实内容不违规 |
| OTC渠道推进慢 | 高 | 低 | 不作为核心GMV来源，作为补充渠道 |
| 法规风险（广告法） | 低 | 高 | 所有内容基于真实体验、避免功效承诺、保留证据 |

### 8.2 最坏情况预案

**如果W7 GMV不到3万：**
1. 暂停所有新渠道尝试，聚焦已验证的最有效获客方式
2. 降价测试（体验装降到29元）验证价格弹性
3. 东东全力发动私域资源
4. 考虑调整目标：从"GMV≥10万"转向"跑通可复制方法论"

**如果团队出现核心成员退出：**
1. 东东兼任空缺角色
2. 简化流程，只保留最核心的动作
3. 优先保GMV和内容产出，其他可以砍掉

### 8.3 底线思维

这个项目最坏的结果是什么？不是GMV不到10万，而是**花了3个月什么数据都没积累**。

只要做到以下三点，这个项目就不会失败：
1. **至少100个真实用户使用过产品并留下了体感数据**——这是最宝贵的资产
2. **至少50篇真实内容发布并积累了数据**——知道什么有效什么无效
3. **团队协作机制跑通了**——方法论和经验可以迁移到下一个项目

GMV是结果，不是目的。目的是**验证商业模型、积累用户资产、锻炼团队能力**。
