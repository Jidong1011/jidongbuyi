## 一、定位与价值主张

### 1.1 一句话定位

**云南第一个以一堂方法论为核心、以产业实战为特色的创业学习中心。**

不搞泛创业社群，不做什么都做的孵化器。只做一件事：帮云南的创业者和中小企业主用一堂五步法（需求→解决方案→商业模式→增长→壁垒）把生意理清楚、做起来。

### 1.2 为什么是一堂，为什么是云南，为什么是现在

**一堂的供给缺口：** 一堂线上覆盖43万人，但线下学习中心全国只有个位数。云南市场完全空白——43万学员中，云南占比估计不到2%（约8000人），但云南实际活跃创业者数量远超这个数字。这不是没有需求，是没有触达渠道。

**云南的市场特征：**
- 大量产业型创业者（农产品、文旅、跨境电商、中药材），有实战经验但缺系统方法论
- 2025年云南GDP增速4.2%，新注册企业超40万户，创业活跃度高
- 省会昆明是西南枢纽，辐射东南亚，但优质创业教育资源极度稀缺
- "云南创业者"自带话题性，能反向为一堂带来品牌势能

**时机窗口：** 2026年商业突破大航海正在进行，东东作为连续三年参与者，个人公信力+社区人脉+产业资源三点齐备。再等半年，其他城市的学习中心可能已经跑通模式，先发优势就没了。

### 1.3 价值主张（对三类人群）

| 人群 | 他们的问题 | 我们给的 |
|------|-----------|---------|
| 产业创业者（农产品/文旅/跨境电商） | 产品好但卖不好，知道做什么但不知道怎么规模化 | 一堂五步法从需求到壁垒的全链路梳理，加云南本地化案例拆解 |
| 初创团队 | 有想法但没验证，团队小资源少 | 轻量级微咨询，帮他们用最小成本完成PMF验证 |
| 一堂线上会员（云南地区） | 线上学完没地方实践、交流、找合伙人 | 线下社群+定期沙龙+本地资源对接 |

### 1.4 差异化竞争点

学习中心不是培训班，不是孵化器，不是商会。它的本质是**方法论落地的本地场域**。三个差异化：

1. **方法论壁垒**——不是"来聊聊生意"，而是用一堂五步法做结构化拆解，这是其他本地社群没有的
2. **产业纵深**——东东在白芸豆产业有真实操盘经验（天保桦、云龙基地），不是纸上谈兵的"导师"
3. **全国网络**——背靠一堂43万学员的全国网络，本地的创业者能对接到全国资源和同频伙伴

## 二、服务产品设计

学习中心有三个产品，形成一个漏斗：沙龙（引流）→ 会员（留客）→ 陪跑（变现）。

### 2.1 活动沙龙——流量入口

沙龙不是目的，是信任建立的起点。设计原则：**低门槛入场，高信息密度输出，明确的下一步引导。**

#### 频次与规模

| 类型 | 频次 | 规模 | 目标 |
|------|------|------|------|
| 周沙龙（轻量） | 每周1-2次 | 15-30人 | 养活场子，持续获客 |
| 月主题沙龙（重磅） | 每月1次 | 50-100人 | 品牌势能，集中转化 |
| 季度产业论坛 | 每季度1次 | 150-300人 | 行业影响力，媒体传播 |

#### 主题矩阵（4大系列）

**系列A：一堂方法论本地拆解**
- 用一堂五步法拆解一个云南本地项目（每次选一个真实案例，最好是参与者自己的项目）
- 示例：「普洱茶新品牌的增长路径——用一堂五步法从需求到壁垒」「一家云南咖啡豆电商的商业模式复盘」
- 频次：月度1次

**系列B：产业创业实战分享**
- 邀请产业创业者做真实分享（不讲概念，只讲踩过的坑和做对的事）
- 示例：「白芸豆产业从田间到货架——一个产业架构师的真实拆解」「跨境电商东南亚市场的0到1」
- 频次：月度1次

**系列C：创业者的第一次碰撞**
- 针对早期创业者，带着想法来，现场用一堂方法论做1小时快速拆解
- 形式：分组工作坊，每组4-6人，东东做教练引导
- 频次：双周1次

**系列D：闭门深度局**
- 针对已有一定规模的企业主，深度探讨一个商业问题
- 示例：「团队管理从10人到50人」「供应链金融怎么做」「品牌出海第一步怎么走」
- 准入条件：报名审核，10-15人
- 频次：月度1次

#### 沙龙运营SOP

每次沙龙的标准化流程：
1. **活动前5天**：发布预告（飞书文档+海报），明确主题、时间、地点、适合谁、你能得到什么
2. **活动前3天**：定向邀请（社群私聊、朋友圈、一对一），目标邀约到场率60%以上
3. **活动前1天**：确认到场人数，准备物料（签到表、方法论卡片、一堂资料袋）
4. **活动中**：90-120分钟，其中内容输出60-70分钟，互动讨论30-40分钟，最后10分钟会员/陪跑转化引导
5. **活动后24小时内**：添加到场者微信（个人号），发送活动复盘总结（飞书文档链接），邀请入社群
6. **活动后7天内**：对高意向参与者做一对一跟进（电话或面聊），推会员或陪跑方案

### 2.2 一堂会员纳新——留存核心

会员纳新不是"卖课"，是"筛选对的人加入正确的社区"。

#### 转化漏斗设计

```
沙龙参与者 → 社群沉淀（免费）→ 线上体验课引导 → 一堂会员转化 → 深度参与
```

具体转化路径：

**第1步：沙龙→社群（转化率目标30%）**
- 每场沙龙结束，扫二维码加入"一堂云南学习社群"
- 进群自动发送欢迎语+一堂方法论一图读懂+3篇精选文章
- 进群后48小时内群主做简短介绍和互动，不让群冷场

**第2步：社群→体验课（转化率目标20%）**
- 在社群内定期推送一堂线上免费体验课/公开课链接
- 每月组织1次"线上课集体看"线下聚会，看完做讨论
- 对参加过2次以上沙龙的群友，做一对一邀约体验

**第3步：体验课→会员（转化率目标25%）**
- 体验课结束后，推送一堂会员权益对比+东东作为"过来人"的使用感受
- 提供限时优惠（如：通过学习中心报名额外赠送1次1对1咨询）
- 关键动作：不是群发海报，是一对一私聊跟进，解答个性化疑问

**第4步：会员→深度参与（复购/推荐率目标40%）**
- 定期组织会员专属线下活动
- 邀请会员做沙龙分享嘉宾（给曝光，给舞台）
- 会员推荐新会员成功，双方各得1次免费沙龙

#### 纳新目标（季度）

| 指标 | Q2目标 | Q3目标 | Q4目标 |
|------|--------|--------|--------|
| 沙龙参与人次 | 300+ | 500+ | 800+ |
| 社群新增人数 | 100+ | 200+ | 300+ |
| 新增一堂会员 | 30-50人 | 60-80人 | 80-120人 |

> 会员收入归一堂平台（学习中心拿返佣），具体分润比例需与一堂总部确认。按行业惯例，线下学习中心推荐会员返佣比例通常在30%-50%之间。假设客单价2000元/年，返佣比例40%，即每单800元。

### 2.3 团队微咨询陪跑——利润中心

这是学习中心真正赚钱的产品。设计原则：**标准化产品、可预期的结果、明确的交付物。**

#### 产品矩阵

| 产品 | 定价 | 周期 | 交付物 | 适合谁 |
|------|------|------|--------|--------|
| **商业诊断局** | 2,000元/次 | 半天（4小时） | 一份商业诊断报告（需求验证、竞品分析、模式建议） | 有产品/项目但卡在某个关键环节的创业者 |
| **1对1创业教练** | 1,500元/次 | 1.5小时 | 纪要+行动计划 | 需要针对性解决单个问题的创业者 |
| **轻陪跑30天** | 8,000元/月 | 30天，每周1次1对1（1小时）+微信群随时答疑 | 商业模式梳理报告+4周行动计划 | 初创团队，需要持续指导但不需重度介入 |
| **深度陪跑90天** | 20,000元/季 | 90天，每周1次1对1（1.5小时）+不限次微信群+参加所有沙龙 | 完整商业计划书+90天执行方案+资源对接 | 有明确项目、需要系统化搭建商业体系的团队 |
| **产业专项咨询** | 面议（3万-10万） | 按项目 | 定制化产业研究报告+商业模式方案+资源对接 | 需要做产业级别规划的企业 |

#### 定价逻辑

- **对比参照：** 一堂线上大航海价格约5,000-15,000元/期；市面上商业咨询顾问日费3,000-8,000元；本地管理咨询项目起步10万
- **我们的定价策略：** 一堂方法论指导的轻咨询，价格是传统咨询的1/3-1/5，但交付物更具体、更可执行
- **核心卖点：** "用一堂五步法帮你理清生意"——方法论有品牌背书，东东有实战经验，价格低到创业者试得起

#### 陪跑客户来源

1. 沙龙参与者中的高意向客户（参加3次以上沙龙→主动提出需要深度帮助）
2. 一堂会员中的升级客户（加入会员后发现需要更个性化指导）
3. 大航海项目中的云南本地团队（天然的信任基础）
4. 白芸豆产业链上下游企业（东东的产业资源转化）

## 三、商业模式与收入模型

### 3.1 收入结构

| 收入来源 | 收入性质 | 毛利率 | 预期占比（稳定期） |
|----------|---------|--------|-------------------|
| 沙龙门票/赞助 | 营收（覆盖成本） | 20-30% | 15% |
| 一堂会员返佣 | 佣金收入 | 95%+ | 25% |
| 微咨询陪跑 | 核心利润 | 80%+ | 50% |
| 企业内训/定制咨询 | 高利润项目 | 70%+ | 10% |

### 3.3 成本结构

| 成本项 | 月费用估算 | 说明 |
|--------|-----------|------|
| 场地（固定） | 3,000-5,000元 | 合作的联合办公空间或咖啡馆包场，非独立租赁 |
| 助理/运营（1人） | 5,000-8,000元 | 兼职或实习生，负责社群运营、活动执行 |

| 差旅/接待 | 1,000-2,000元 | 拜访客户、嘉宾接待 |
| **月合计** | **12,000-20,000元** | |
| **年合计** | **14.4万-24万** | |

## 四、目标人群与获客策略

### 4.1 核心目标人群画像

**人群A：产业创业者（占比40%，最核心）**
- 年龄：28-45岁
- 行业：农产品加工、文旅/民宿、跨境电商、中药材、特色餐饮
- 特征：有产品、有供应链、有产业认知，但商业模式模糊，增长乏力
- 痛点："我东西不差，就是不知道怎么规模化""线上怎么搞""怎么找到对的人"

**人群B：早期创业者（占比30%）**
- 年龄：25-35岁
- 行业：不限定，多为服务业/科技/内容创业
- 特征：刚起步或准备创业，有想法但未验证，容易焦虑
- 痛点："不知道第一步该做什么""怕选错方向""一个人太孤独"

**人群C：中小企业主/转型升级者（占比20%）**
- 年龄：35-50岁
- 行业：传统行业寻求转型
- 特征：企业有稳定收入但增长见顶，需要找到第二曲线
- 痛点："传统业务越来越难做""看到别人做电商/出海/品牌也想做但不知道从哪入手"

**人群D：一堂在云南的存量会员（占比10%）**
- 已经是一堂线上会员，需要线下场域交流和实践
- 高潜力人群：有付费意愿，需要的是线下归属感和深度服务

### 4.2 获客渠道

| 渠道 | 策略 | 预期效果 | 成本 |
|------|------|---------|------|
| **东东个人IP** | 抖音/视频号/小红书持续输出创业方法论+产业洞察内容，引流到私域 | 长期最大流量源，前期月增50-100粉 | 时间成本为主 |
| **一堂社区** | 在大航海社群/会员群中建立"云南学习中心"认知，吸引云南学员 | 精准流量，转化率高 | 低（已有信任基础） |
| **线下沙龙口碑** | 每场沙龙要求到场者"带一个朋友"，制造裂变 | 口碑传播最有效的方式 | 极低 |
| **产业圈层** | 白芸豆产业、茶产业、咖啡产业——东东已有的产业人脉 | 高质量客户，转化周期短 | 社会资本 |
| **异业合作** | 与本地的创业园区、孵化器、商会、校友会合作办活动 | 批量获客 | 中（需要BD时间） |
| **飞书社群+朋友圈** | 东东朋友圈运营+学习中心飞书社群 | 稳定的激活和复购 | 低 |

### 4.3 获客节奏

- **第1-2个月：** 以一堂社区+产业人脉为主，精准邀约，不求量求质
- **第3-4个月：** 启动个人IP内容输出，配合沙龙口碑裂变
- **第5-6个月：** 开展异业合作，扩大漏斗入口

## 五、运营节奏

### 5.1 每周节奏

| 时间 | 动作 |
|------|------|
| 周一 | 发布本周沙龙预告，定向邀约 |
| 周二/周三 | 社群内容输出（方法论小知识/案例拆解），朋友圈互动 |
| 周四晚 | 周沙龙（轻量） |
| 周六下午 | 周沙龙（轻量）或1对1咨询 |
| 周日 | 复盘本周数据，下周规划，内容准备 |

### 5.2 月度节奏

| 周次 | 重点 |
|------|------|
| 第1周 | 月度主题沙龙筹备，当月陪跑项目1对1启动 |
| 第2周 | 月度主题沙龙举办，大力度纳新 |
| 第3周 | 闭门深度局或产业专题沙龙，陪跑项目中期检查 |
| 第4周 | 月度复盘，数据统计，下月规划，个人IP内容集中产出 |

### 5.3 季度里程碑

| 季度 | 核心目标 | 关键事件 |
|------|---------|---------|
| Q2（4-6月） | 冷启动，跑通模式 | 举办首场沙龙，首批30+社群成员，首个陪跑客户 |
| Q3（7-9月） | 规模化复制 | 稳定周沙龙频次，社群100+人，季度论坛首秀 |
| Q4（10-12月） | 商业验证 | 总收入达到30万+，跑通一个产业专题（白芸豆或茶），确认可持续模式 |

## 六、启动计划（前90天行动清单）

### 第1-30天：基建期——"从0到0.5"

**场地：**
- [ ] 确定常驻活动场地（优先级：联合办公空间 > 书店/咖啡馆 > 创业园区）
- [ ] 场地预算：月包场费3,000-5,000元或按次结算200-500元/场
- [ ] 备选：昆明市区2-3个可用的合作场地

**品牌基建：**
- [ ] 注册微信公众号/服务号"一堂云南学习中心"
- [ ] 设计学习中心LOGO和视觉VI（可简版，飞书文档模板即可）
- [ ] 制作一套标准物料：方法论卡片、签到表、活动海报模板
- [ ] 建立飞书社群，邀请首批种子用户（20-30人）

**内容准备：**
- [ ] 准备3套沙龙PPT（方法论拆解、产业案例、互动工作坊）
- [ ] 撰写3篇首发文章（"为什么在云南开一个学习中心""一堂五步法是什么""我的大航海三年"）

## 八、与大航海项目的协同效应

### 8.1 大航海能给学习中心什么

1. **信任背书**——"连续三年大航海参与者"是一堂社区内的高信誉标签，学习中心天然获得社区信任
2. **内容输入**——大航海每期的最新方法论、案例、工具，可以直接转化为沙龙内容
3. **人才供给**——大航海中的云南本地成员，是学习中心最精准的潜在客户和合作伙伴
4. **品牌曝光**——在一堂大航海社群中分享学习中心的进展，获得全国性的品牌曝光

### 8.2 学习中心能反馈给大航海什么

1. **地域数据**——云南创业者的真实需求、痛点、案例，为一堂方法论迭代提供区域样本
2. **内容输出**——学习中心产出的本地案例拆解、产业深度内容，可以反哺一堂内容库
3. **线下实验场**——一堂需要验证新课程/新方法论时，云南学习中心可以作为线下测试场
4. **人才输送**——优秀的云南创业者，通过学习中心推荐进入大航海，提升大航海的生源质量

### 8.3 协同节奏

| 时间 | 协同动作 |
|------|---------|
| 大航海报名期 | 学习中心定向邀约云南创业者参加，作为学习中心的"入学通道" |
| 大航海进行中 | 组织"大航海线下讨论会"，让参加大航海的云南学员线下碰撞 |
| 大航海结束后 | 举办"大航海成果分享会"，分享学员的实战收获，同时转化新会员 |

## 九、风险与对冲

| 风险 | 发生概率 | 影响 | 对冲策略 |
|------|---------|------|---------|
| **获客困难，沙龙到场率低** | 中 | 高 | 控制单场成本（<500元），不靠单场沙龙赚钱，靠沙龙后的私域转化。前5场以"定向邀请"代替"开放报名" |
| **陪跑产品卖不动** | 中 | 高 | 先用低价"商业诊断局"做测试，验证需求。如果3个月0客户，说明定价或定位有问题，需要调整 |
| **一堂总部不支持** | 低 | 极高 | 提前沟通，确认合作框架。如果品牌授权受限，可先以"纪东不已创业学习中心"名义运营，用一堂方法论但不用一堂品牌 |
| **东东时间不够** | 高 | 高 | 严格按优先级分配时间：陪跑客户>沙龙准备>个人IP>社群运营。招募助理分担运营工作。控制沙龙频次，不追求数量 |
| **同质化竞争** | 低 | 中 | 一堂方法论的稀缺性构成壁垒。即使本地出现类似社群，东东的大航海背景和产业深度难以复制 |
| **收入不及预期** | 中 | 中 | 前期不依赖学习中心收入养活自己，把学习中心定位为"1年期创业项目"而非"立即赚钱的生意"。给6个月验证期 |
| **个人IP做不起来** | 中 | 中 | 个人IP是加分项不是必选项。核心获客靠沙龙+产业人脉，IP做不起来影响增长速度但不影响生存 |
